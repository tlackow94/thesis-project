Project Goals and Objectives

Goals

  • Assist with identifying solutions to help improve renewable energy resources in California.
  • Expanding upon existing knowledge base regarding new renewable energy developments.
  • Provide a robust framework that other organizations can utilize.

Objectives

  • Determine framework for ranking locations in terms of suitability.
  • Develop a database of potential locations for developing renewable resources.
  • Create a map of locations, ranked by suitability.
  • Create an interactive dashboard for users to access.

Data Sources

The terrain data is derived from 1-meter DEM retrieved from The National Map supplied by the United States Geological Survey.

Methods

This project utilizes Multiple-Criteria Decision Analysis (MCDA) as the foundation. MCDA is utilized in site selection when there are multiple conflicting criteria. Traditionally, each criteria is ranked depending upon its range of values, with some values prioritized over others. A site's final score is a summation of the rankings from the various criteria that has been identified.

The terrain criteria for this project include:

  • Slope
  • Aspect
  • Parcel Ownership
